# Snake 3D — UI 设计规范
> 版本：1.0 | 日期：2026-04-02
> 设计风格：Cyberpunk / Neon 沉浸式

---

## 一、核心设计原则

1. **沉浸感第一** — 3D 场景是主角，UI 只是光效点缀
2. **操作直觉** — 鼠标跟随 = 0 学习成本
3. **反馈强烈** — 吃食物/死亡都有电影级视觉冲击
4. **炫耀有面** — 死亡后一键分享截图

### 3.2 游戏 HUD（Playing）

```
┌──────────────────────────────────────────┐
│ [💰 12,450]              [⏱️ 01:23]       │
│                                          │
│                                          │
│              (3D 游戏场景)                │
│                                          │

│                                          │
│                                          │
│                                          │
│ [📍 段位：黄金]        [🎯 目标：5000]    │
└──────────────────────────────────────────┘
```

**元素说明：**
- 左上：当前金币（吃食物 +1，存活30秒 +10）
- 右上：存活时间
- 左下：当前段位（青铜→白银→黄金→钻石→王者）
- 右下：本局目标分数

**视觉效果：**
- 半透明黑色背景条（rgba(0,0,0,0.3)）
- 文字有微弱发光
- 数字变化时有跳动动画

## 四、交互动效规范

| 动作 | 动效描述 |
|------|----------|
| 按钮 Hover | 0.2s scale(1.05) + 发光增强 |
| 按钮点击 | 0.1s scale(0.95) + 电流特效 |
| 分数增加 | 数字弹跳 + 发光闪烁 |
| 新纪录 | 金色粒子爆炸 + 屏幕金光 |
| 死亡 | 蛇身碎片化 + 慢镜头 1s |
| 界面出现 | 从中心放大 + 淡入 0.3s |
| 界面消失 | 缩小淡出 0.2s |

## 五、技术实现

### 字体
- 主字体：Orbitron（Google Fonts）- 科幻感
- 数字字体：Share Tech Mono - 等宽清晰

### 图标
- 使用 emoji 作为主要图标（零成本、高兼容性）
- 复杂图标可用 Lucide Icons

### 实现方式
- HTML/CSS/JavaScript 独立完成
- CSS Variables 管理主题色
- CSS Animations + Web Animations API
