## 1. 游戏概述

### 1.1 视觉风格
参考"贪食蛇之王"风格：
- **背景**: 紫粉渐变 + 装饰星星/光点
- **蛇**: 鲜艳纯色 + 可爱表情
- **UI**: 圆润卡片、阴影、高对比度

### 1.2 目标用户
- 休闲玩家（8-45岁）
- 喜欢可爱卡通风格
- 追求视觉冲击和社交展示

## 3. 游戏区域设计

### 3.1 画布布局
```
┌─────────────────────────────────────┐
│  [玩家头像] SCORE: 1250   [设置]    │  ← 顶部状态栏
├─────────────────────────────────────┤
│                                     │
│     ┌───────────────────────┐       │
│     │                       │       │
│     │     游戏画布区域       │       │  ← 主游戏区（圆角矩形）
│     │     (深蓝网格背景)     │       │
│     │                       │       │
│     └───────────────────────┘       │
│                                     │
│  [模式选择: Classic ▼]              │  ← 底部模式选择
└─────────────────────────────────────┘
```

### 3.2 游戏画布样式
- **背景**: 深蓝色 #1a1a4e 带细网格线 #2a2a6e
- **边框**: 8px 圆角, 金色边框 #FFD700, 发光效果
- **尺寸**: 桌面 500×500, 移动端 自适应

### 3.3 蛇设计
- **身体**: 圆角矩形节段，相邻颜色微妙渐变
- **头部**: 比身体大一点，有两个眼睛（跟随方向）
- **尾巴**: 逐渐变小的圆点
- **默认蛇**: 明黄色 #FFD700
- **备用蛇**: 鲜红色 #FF4444

### 3.4 食物设计
- **形状**: 苹果/浆果（CSS/SVG绘制）
- **颜色**: 红色 #FF6B6B 带绿色叶子
- **动画**: 轻微弹跳 + 发光脉冲

## 5. 游戏模式

### 5.1 Classic Mode（经典模式）
- 无时间限制
- 无道具
- 吃食物得分 (每个10分)
- 撞墙/撞自己 = 结束

### 5.2 Time Attack（限时模式）
- 60秒倒计时
- 每15秒速度加快
- 显示剩余时间

### 5.3 Obstacle（障碍模式）
- 随机障碍物出现
- 障碍物碰撞 = 结束

---

## 6. 道具系统

### 6.1 道具类型
| 道具 | 符号 | 颜色 | 效果 |
|------|------|------|------|
| 加速 | ⚡ | 金色 | 速度+50%, 5秒 |
| 护盾 | 🛡️ | 蓝色 | 免死一次 |
| 双倍 | ⭐ | 粉色 | 分数×2, 10秒 |
| 巨型食物 | 🍎 | 红色 | 一个食物=50分 |

### 6.2 道具生成
- 每吃3个食物有40%概率
- 道具存在10秒后消失
- 道具出现时播放动画

---

## 7. 动效系统

### 7.1 粒子效果
- **吃食物**: 彩色粒子爆发（8-12个粒子向外扩散）
- **道具获取**: 星形粒子旋转上升
- **游戏结束**: 屏幕震动 + 红色闪烁

### 7.2 UI 动效
- **按钮悬停**: 轻微放大 + 阴影加深
- **分数增加**: 数字弹跳动画 + 闪光
- **菜单出现**: 从底部滑入 + 弹性

### 7.3 装饰元素
- **背景星星**: 随机位置 + 闪烁动画
- **浮动小蛇**: 屏幕边缘飘动的小蛇轮廓
- **云朵**: 缓慢飘动的装饰云
