## 1. 游戏概述

### 1.1 游戏类型
经典贪吃蛇游戏的现代化升级版，融合霓虹赛博朋克视觉风格。

### 1.2 目标用户
- 休闲玩家（15-45岁）
- 喜欢经典游戏翻新
- 追求视觉享受和流畅体验

### 1.3 核心玩法
- 控制蛇移动，吃食物得分
- 躲避墙壁和自己的尾巴
- 收集道具获得特殊能力
- 解锁皮肤展示成就

---

## 2. 游戏模式

### 2.1 Classic Mode（经典模式）
- **目标**: 吃尽可能多的食物
- **失败条件**: 撞墙或撞自己
- **计分**: 每个食物 = 10分
- **特殊**: 无时间限制，无道具

### 2.2 Time Attack（限时挑战）
- **目标**: 60秒内获得最高分
- **计时**: 60秒倒计时
- **加速**: 每15秒速度提升20%
- **计分**: 每个食物 = 10分，×2加速时 = 20分

### 2.3 Obstacle（障碍模式）
- **目标**: 躲避随机生成的障碍物
- **障碍物**: 每吃3个食物生成一个
- **失败条件**: 撞墙/撞自己/撞障碍物
- **计分**: 每个食物 = 15分

## 3. 道具系统

### 3.1 道具类型
| 道具 | 符号 | 效果 | 持续时间 | 出现概率 |
|------|------|------|----------|----------|
| 加速 | ⚡ | 移动速度+50% | 5秒 | 15% |
| 护盾 | 🛡️ | 免疫一次碰撞 | 一次 | 10% |
| 双倍 | ⭐ | 分数×2 | 10秒 | 15% |
| 巨型食物 | 🍎 | 食物=50分 | 吃到为止 | 8% |
| 减速 | 🐌 | 移动速度-30% | 5秒 | 12% |

### 3.2 道具生成规则
- 基础生成间隔：每吃掉3个食物有40%概率生成道具
- 道具存在时间：10秒后消失
- 道具激活时间：2秒后可用（防止立即碰撞）

## 4. 蛇皮肤系统

### 4.1 皮肤列表
| 皮肤 | 解锁条件 | 颜色渐变 |
|------|----------|----------|
| 经典绿 | 默认解锁 | #00ff88 → #009944 |
| 烈焰红 | 500分 | #ff6600 → #cc0000 |
| 冰霜蓝 | 1000分 | #00ffff → #0066ff |
| 银河紫 | 2000分 | #ff00ff → #9900ff |
| 彩虹光 | 5000分 | 全彩渐变动画 |

### 4.2 皮肤显示
- 蛇头始终有眼睛，方向跟随移动
- 身体为圆角矩形渐变色
- 皮肤解锁后自动装备

## 5. UI/UX 设计

### 5.1 界面流程
```
[启动页] → [主菜单] → [模式选择] → [游戏中] → [游戏结束] → [主菜单/重玩]
                ↑                    ↓
                ←←←←←←←←←←←←←←←←←
```

### 5.2 屏幕设计

#### 5.2.1 主菜单
- 游戏标题（霓虹发光效果）
- "PLAY" 按钮（大、醒目）
- "HIGH SCORES" 按钮
- "SETTINGS" 按钮
- 背景：动态粒子效果

#### 5.2.2 模式选择
- 三个模式卡片（Classic / Time Attack / Obstacle）
- 每个卡片显示模式名称、图标、简短描述
- 当前最高分显示

#### 5.2.3 游戏界面
- 顶部：当前分数 | 模式 | 道具状态
- 中央：游戏画布
- 底部（移动端）：方向触控区

#### 5.2.4 游戏结束
- "GAME OVER" 或 "TIME'S UP"
- 最终分数（大字体）
- 最高分比较
- "RETRY" / "MENU" 按钮

### 5.3 视觉风格
- **主题**: 霓虹/赛博朋克
- **背景**: 深色 (#0a0a1a) + 网格线 (#1a1a3a)
- **强调色**: 霓虹绿 (#00ff88)、霓虹粉 (#ff0066)、霓虹青 (#00ffff)
- **字体**: 系统无衬线字体 (Segoe UI / SF Pro / Roboto)

### 5.4 动画效果
- 食物：脉冲发光动画
- 道具：旋转 + 发光
- 蛇移动：平滑插值（可选）
- 碰撞：屏幕闪烁红色
- 吃食物：爆炸粒子效果

## 6. 音效系统

### 6.1 音效列表
| 音效 | 触发时机 | 备注 |
|------|----------|------|
| BGM | 游戏全程循环 | 可开关 |
| eat | 吃食物 | 短促 |
| powerup | 获得道具 | 上升音调 |
| die | 游戏失败 | 下降音调 |
| click | 按钮点击 | 短促 |

### 6.2 实现
- 使用 Web Audio API
- 音效可单独开关
- BGM 可单独开关
- 默认关闭（尊重用户）
