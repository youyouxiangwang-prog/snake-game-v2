## 1. 愿景 (Vision)

打造一款 **真正的 3D 低多边形风格贪食蛇游戏**，具有：

- **沉浸式 3D 世界**：Three.js 渲染的立体世界
- **流畅的相机系统**：跟随蛇移动，动态滚动视角
- **生动的角色动画**：特别是蛇的分叉舌头动画
- **复古怀旧美感**：90年代末/2000年代初的低多边形风格
- **简单但有趣**：易上手，有深度

**核心体验**：让玩家感觉在操控一条真实的 3D 蛇在草地世界中探索。

## 2. 技术原则 (Technical Principles)

### 2.1 3D 引擎
- **Three.js** - Web 原生 3D 库，生态完善
- **纯 JavaScript** - 无需 TypeScript，降低复杂度
- **Vite** - 开发服务器和构建工具

### 2.2 性能优先
- **60 FPS** - 流畅的游戏体验
- **低多边形模型** - 面数少，性能好
- **对象池** - 复用 3D 对象，减少 GC
- **请求动画帧** - 使用 requestAnimationFrame

### 2.3 响应式设计
- **移动端支持** - 触摸控制
- **自适应分辨率** - 根据设备调整渲染尺寸
- **键盘/触摸双支持** - 统一的输入抽象

## 4. 相机系统原则 (Camera System)

### 4.1 核心规则

```
规则 1: 相机必须平滑跟随蛇头
规则 2: 蛇头始终保持在画面中心偏下
规则 3: 相机朝向蛇的移动方向
规则 4: 支持平滑的视角过渡
```

### 4.2 相机参数
```javascript
const CAMERA_CONFIG = {
    // 偏移：从蛇头到相机的向量
    offset: { x: 0, y: 15, z: 15 },
    
    // 插值系数 (0-1，越小越慢)
    lerpFactor: 0.08,
    
    // 前方预看距离
    lookAhead: 5,
    
    // 上下角度 (俯视角度)
    angle: Math.PI / 4  // 45度俯视
};
```

### 4.3 跟随算法
```
1. 计算目标相机位置 = 蛇头位置 + 偏移
2. 平滑插值当前相机位置到目标位置
3. 计算蛇头前方某点作为看点
4. 相机 lookAt 该点
```

## 5. 蛇的设计原则 (Snake Design)

### 5.1 身体结构
```
蛇头 ──► 蛇身段1 ──► 蛇身段2 ──► ... ──► 蛇尾
(带头部   (普通    (普通
 特征)    身体)    身体)
```

### 5.2 蛇头规格
| 部件 | 几何体 | 尺寸 |
|------|--------|------|
| 头部主体 | 扁圆柱 | radius 0.6, height 0.8 |
| 眼睛 | 球体 x2 | radius 0.15 |
| 瞳孔 | 球体 x2 | radius 0.08 |
| 舌头 | V形细长三角 | length 0.8-1.5 |

### 5.3 蛇身规格
| 部件 | 几何体 | 尺寸 |
|------|--------|------|
| 身体段 | 圆角圆柱 | radius 0.4-0.5, height 0.7 |
| 连接处 | 球体 | radius 0.5 |
| 间距 | - | 0.6 单位 |

### 5.4 移动动画
```
蛇身段跟随算法：
1. 蛇头直接跟随输入移动
2. 每节蛇身追踪前一节之前的位置
3. 使用 lerp 插值实现平滑跟随
4. 延迟递增：越靠后的段延迟越大
```

## 6. 舌头动画原则 (Tongue Animation)

### 6.1 动画状态机
```
        ┌─────────┐
        │  IDLE   │◄────────────────┐
        └────┬────┘                 │
             │ 时间触发 (每3-5秒)   │
             ▼                     │
    ┌────────────────┐             │
    │  EXTENDING    │             │
    └────┬──────────┘             │
         │ 达到最大长度            │
         ▼                       │
    ┌────────────────┐            │
    │  RETRACTING   │────────────┘
    └────────────────┘  缩回最小长度
```

### 6.2 舌头参数
```javascript
const TONGUE_CONFIG = {
    // 动画速度
    extendSpeed: 0.15,      // 伸出速度 (单位/帧)
    retractSpeed: 0.2,      // 缩回速度
    
    // 长度范围
    maxLength: 1.2,         // 最大伸出长度
    minLength: 0.1,          // 收起长度
    
    // 形态
    forkAngle: 0.3,         // 分叉角度 (弧度)
    width: 0.06,            // 舌头宽度
    
    // 特效
    color: 0xFF3333,        // 红色
    emissive: 0xFF0000,      // 发光红色
    
    // 触发
    idleDuration: [3, 5],   // 空闲持续时间范围 (秒)
    alwaysOut: true         // 是否总是伸出 (参考图样式)
};
```

### 6.3 渲染要求
```
V形分叉舌头：
- 两个细长三角形的组合
- 从蛇头前端伸出
- 底部合并，顶部分开
- 轻微的左右摇摆动画
```

## 7. 环境设计原则 (Environment)

### 7.1 世界设定
```
世界类型：草地荒野
世界大小：100 x 100 单位
边界：不可见的墙
```

### 7.2 地面
| 属性 | 值 |
|------|-----|
| 几何体 | PlaneGeometry 100x100 |
| 分段 | 50x50 (用于高度变化) |
| 材质 | MeshLambertMaterial, flatShading |
| 颜色 | 深绿/浅绿棋盘格或渐变 |

### 7.3 障碍物
| 类型 | 几何体 | 颜色 |
|------|--------|------|
| 树桩 | CylinderGeometry (截断) | 棕色 |
| 石块 | DodecahedronGeometry | 灰色 |
| 随机生成 | 每 20-30 单位一个 | - |

### 7.4 食物
| 属性 | 值 |
|------|-----|
| 类型 | 苹果 |
| 几何体 | SphereGeometry + ConeGeometry(叶) + CylinderGeometry(茎) |
| 颜色 | 红色 + 绿色 |
| 生成 | 随机位置，不与蛇/障碍重叠 |

## 8. 游戏玩法原则 (Gameplay)

### 8.1 核心循环
```
1. 蛇自动向前移动
2. 玩家控制方向 (上下左右)
3. 吃苹果得分，蛇身变长
4. 撞墙/撞自己/撞障碍物 = 游戏结束
```

### 8.2 游戏模式
| 模式 | 描述 |
|------|------|
| Classic | 无限时间，无尽模式 |
| Time Attack | 限时60秒，比速度 |
| Obstacle | 增加随机障碍物 |

### 8.3 计分
```
+10 分：吃一个苹果
+5 分：每存活10秒 (Time Attack)
```
