# Constitution - Snake Game V2 (Colorful Cartoon Edition)

**Project**: Snake Game V2
**Style**: Colorful Cartoon / Chinese Mobile Game Aesthetic
**Created**: 2026-03-25

---

## Global Rules

### 技术栈
- **前端**: 纯 HTML5 + CSS3 + Vanilla JavaScript
- **渲染**: Canvas 2D API
- **构建**: 单一 HTML 文件，可直接运行
- **资源**: 内联 SVG / CSS 绘制（无外部图片依赖）

### 代码规范
- **变量/函数**: 小驼峰 (camelCase)
- **常量**: 大写下划线 (UPPER_SNAKE_CASE)
- **类名**: 大驼峰 (PascalCase)
- **文件名**: 小写 + 连字符 (kebab-case.html)

### 项目结构
```
snake-game-v2/
├── index.html          # 入口（单一文件）
├── SPEC.md             # 项目规范
├── .specify/
│   ├── CONSTITUTION.md # 宪法
│   ├── SPEC.md         # 需求规范
│   ├── CLARIFY.md      # 澄清
│   └── PLAN.md         # 实施计划
└── assets/             # 静态资源
```

### 核心原则
1. **色彩优先**: 高饱和度、鲜艳、充满活力
2. **卡通风格**: 圆润、可爱、Q弹
3. **动效丰富**: 粒子、闪烁、弹跳
4. **社交元素**: 分数榜、皮肤展示

### 禁止事项
- ❌ 暗色系/赛博朋克风格
- ❌ 外部框架依赖
- ❌ 跳过澄清直接实现
